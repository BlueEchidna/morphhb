MorphHB is a project to analyze the Hebrew Bible by lemma and morphology. 
The project is marked up in OSIS XML and currently contains lemma attributes 
for most words (using Strong's numbers). We plan to add morphology attributes 
as well.

The text in the download section, WLC.4.12.zip, is released to the public domain 
(http://creativecommons.org/licenses/publicdomain/).
